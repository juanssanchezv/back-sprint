# TicDrive-Back-End

Back End desarrollado en python-Fast Api para proyecto TicDrive 
- Para consumir esta api use este Front End -> https://github.com/cloudmonsterrepo/TicDrive-Front-End

## Build Setup

``` bash
# instalacion de dependencias
pip install -r requirements.txt
o
pip3 install -r requirements.txt

# Iniciar host localhost:8000
uvicorn main:app --reload

```
